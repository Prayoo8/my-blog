3. 检查主题是否影响高亮设置

有些 Hexo 主题（如 NexT）可能自带独立的高亮配置，与 Hexo 根目录的 _config.yml 配置无关。

检查主题配置文件：

打开主题目录下的 _config.yml 文件（如 themes/next/_config.yml），找到与代码高亮相关的部分：

codeblock:
  # 是否启用内置的代码高亮样式
  enable: true
  # 高亮主题，替换为你需要的主题
  highlight_theme: solarized-light
  # 是否启用行号
  line_number: true
